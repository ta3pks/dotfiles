  <name>Create test_bootstrap.sh test suite</name>

  <action>
Create `test_bootstrap.sh` at repo root. Make it executable (`chmod +x`).

**Test framework:**
- Simple bash test harness: `assert_eq`, `assert_file_exists`, `assert_symlink`, `assert_not_exists` helpers
- Each test function prefixed with `test_`
- Track pass/fail counts, print summary at end
- Exit 0 if all pass, exit 1 if any fail
- Use colored output (green PASS, red FAIL)

**Test environment setup:**
- Create temp directory (`mktemp -d`) as fake HOME
- Set `HOME` to temp dir for each test
- Create `~/.config` dir in temp HOME
- Clean up temp dir after each test (trap on EXIT)
- Tests run against the REAL repo (read-only) but write symlinks to temp HOME

**Test cases:**

**AUTO-01: Script runs without errors**
1. `test_help_flag` — `./bootstrap.sh --help` exits 0
2. `test_essential_noninteractive` — `./bootstrap.sh --essential --conflict=skip` exits 0 with temp HOME
3. `test_missing_prereq_fails` — Mock missing jq by prepending empty dir to PATH, verify exit 2

**AUTO-02: Creates all necessary symlinks**
4. `test_essential_symlinks_created` — Run with --essential, verify each essential tool's symlinks exist via `readlink`
5. `test_directory_symlink_type` — Verify nvim creates directory-level symlink (`~/.config/nvim` → repo)
6. `test_home_dotfile_type` — Verify git creates home file symlinks (`~/.gitconfig` → repo)
7. `test_nested_symlink_type` — Verify local_bin creates individual file symlinks in `~/.local/bin/`

**AUTO-03: Idempotent**
8. `test_idempotency` — Run bootstrap twice with same args. Second run should report all "already linked", no errors. Diff filesystem state (find + readlink) — must be identical.
9. `test_conflict_backup` — Create a fake file at a target location, run with --conflict=backup. Verify .bak exists AND symlink created.
10. `test_conflict_skip` — Create a fake file at target, run with --conflict=skip. Verify original file untouched, no symlink.

**Undo tests:**
11. `test_undo` — Run bootstrap, then --undo. Verify symlinks removed. Verify .bootstrap-manifest removed.
12. `test_undo_restores_backups` — Create fake file, run with --conflict=backup, then --undo. Verify .bak restored to original name.

**Dry-run tests:**
13. `test_dry_run_no_changes` — Run with --dry-run. Verify NO symlinks created, NO .bootstrap-manifest written. Verify output contains "Would link".

**Output format:**
```
=== Bootstrap Test Suite ===
[PASS] test_help_flag (AUTO-01)
[PASS] test_essential_noninteractive (AUTO-01)
[FAIL] test_missing_prereq_fails (AUTO-01) - Expected exit 2, got 0
...
=== Results: 12/13 passed ===
```
