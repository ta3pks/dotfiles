<tasks>

<task type="auto">
  <name>Task 1: Install gitleaks and scan repository</name>
  <files>scripts/audit-secrets.sh</files>
  <action>
    1. Check if gitleaks is installed: `which gitleaks || command -v gitleaks`
    2. If not installed, install via: `brew install gitleaks` (macOS) or download binary from https://github.com/gitleaks/gitleaks/releases
    3. Create `scripts/audit-secrets.sh` that runs: `gitleaks git --verbose .`
    4. Run the scan on the entire repository history: `gitleaks git --verbose --report-path .planning/gitleaks-report.json .`
    5. Review findings - legitimate secrets must be removed, false positives go to .gitleaksignore
    
    **Scan scope per CONTEXT.md:**
    - API keys, tokens, passwords
    - Private keys (SSH, GPG)
    - AI tool configs (opencode/.config/opencode/, .aider*)
    - Environment files (.env, .env.*)
    - Shell history files
    - Credential files (credentials.json, secrets.yaml)
    
    **Exclude:** Public keys, known-safe files
  </action>
  <verify>
    <automated>gitleaks git --verbose . 2>&1 | grep -q "no leaks found" && echo "PASS: No secrets detected" || echo "FAIL: Secrets found - review report"</automated>
    <manual>Review .planning/gitleaks-report.json if created for any findings</manual>
    <sampling_rate>Run after this task commits</sampling_rate>
  </verify>
  <done>Gitleaks scan completes with exit code 0 (no secrets) or report generated for review</done>
</task>

<task type="auto">
  <name>Task 2: Create baseline for acceptable findings</name>
  <files>.gitleaksignore</files>
  <action>
    If gitleaks found legitimate false positives (test data, example keys, etc):
    1. Review .planning/gitleaks-report.json
    2. For each false positive, extract the Fingerprint from the report
    3. Add fingerprints to .gitleaksignore file at repo root
    4. Re-run scan to verify baseline works: `gitleaks git --verbose .`
    
    If no false positives, create empty .gitleaksignore with comment:
    ```
    # Gitleaks ignore file
    # Add fingerprints of acceptable findings here if needed
    # Format: one fingerprint per line
    ```
  </action>
  <verify>
    <automated>test -f .gitleaksignore && echo "PASS: Baseline file exists"</automated>
    <sampling_rate>Run after this task commits</sampling_rate>
  </verify>
  <done>.gitleaksignore file exists at repo root,done>
</task>
