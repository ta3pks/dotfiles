# Plan 01: Audit Repository for Hardcoded Secrets

<objective>
Scan the entire repository for hardcoded secrets using gitleaks to ensure no credentials are exposed in version control.
</objective>

<context>
@.planning/phases/01-security-audit/01-RESEARCH.md
@.planning/PROJECT.md
</context>

<tasks>

<task type="auto">
  <name>Install gitleaks for secret scanning</name>
  <files></files>
  <action>
    Install gitleaks using the recommended method for Linux:
    ```bash
    # Download latest gitleaks release
    cd /tmp
    curl -sSfL https://github.com/gitleaks/gitleaks/releases/latest/download/gitleaks_linux_amd64.tar.gz | tar -xz
    sudo mv gitleaks /usr/local/bin/
    sudo chmod +x /usr/local/bin/gitleaks
    ```
  </action>
  <verify>`gitleaks version` returns version string</verify>
  <done>Gitleaks installed and available in PATH</done>
</task>

<task type="auto">
  <name>Scan repository for secrets</name>
  <files></files>
  <action>
    Run gitleaks scan on the entire git history:
    ```bash
    cd /home/nikos/dotfiles
    gitleaks git --verbose --report-path .planning/phases/01-security-audit/gitleaks-report.json 2>&1 || true
    ```
    Capture exit code and any findings.
  </action>
  <verify>Scan completes (exit code 0 = no secrets, exit code 1 = secrets found)</verify>
  <done>Full repository scan completed, report generated</done>
</task>

<task type="auto">
  <name>Review and document findings</name>
  <files>.planning/phases/01-security-audit/gitleaks-report.json</files>
  <action>
    If secrets found:
    1. Review the gitleaks report
    2. Document findings in a summary
    3. If any real secrets found, they must be removed from git history
    
    If no secrets found:
    1. Document clean scan result
    2. Proceed to next plan
  </action>
  <verify>Findings documented or clean scan confirmed</verify>
  <done>Security audit complete, findings documented</done>
</task>

</tasks>

<verification>
- Gitleaks installed and working
- Full repository scan completed
- Report generated at .planning/phases/01-security-audit/gitleaks-report.json
- No hardcoded secrets in version control (or documented remediation plan)
</verification>

<success_criteria>
- [ ] Gitleaks installed
- [ ] Repository scanned
- [ ] Report generated
- [ ] No secrets exposed (or remediation plan documented)
</success_criteria>
