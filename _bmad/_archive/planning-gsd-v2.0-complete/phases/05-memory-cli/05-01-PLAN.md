# Phase 5 Plan 01: CLI Core Commands

**Wave:** 1
**Depends on:** Phase 4 (Storage Foundation)
**Files modified:** memory/src/cli/*.ts, memory/package.json
**Autonomous:** true
**Requirements:** CLI-01, CLI-02, CLI-03, CLI-04, CLI-05

## Goal

Build the `memory` CLI binary with core commands: store, search, get, list, delete. Users can interact with the memory system from the command line with human-readable and JSON output.

## must_haves

- [ ] `memory store` creates memories with auto-embedding
- [ ] `memory search` performs semantic search
- [ ] `memory get` retrieves by ID
- [ ] `memory list` browses by tag/project
- [ ] `memory delete` removes memories
- [ ] JSON output via `--json` flag
- [ ] Proper exit codes (0/1/2/3)
- [ ] Help text for all commands
