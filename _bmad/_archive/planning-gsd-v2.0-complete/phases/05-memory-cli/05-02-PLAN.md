# Phase 5 Plan 02: Memory Type Handlers

**Wave:** 2
**Depends on:** 05-01-PLAN.md
**Files modified:** memory/src/cli/types/*.ts, memory/src/cli/commands/store.ts
**Autonomous:** true
**Requirements:** MEM-01, MEM-02, MEM-03, MEM-04, MEM-05

## Goal

Add memory type system with specialized handling for patterns, decisions, context, knowledge, and preferences. Each type has type-specific fields and validation.

## must_haves

- [ ] `--type pattern` stores code patterns with language field
- [ ] `--type decision` stores decisions with rationale field
- [ ] `--type context` stores conversation context
- [ ] `--type knowledge` stores cross-project knowledge
- [ ] `--type preference` stores user preferences
- [ ] Type-specific prompts in interactive mode
- [ ] Type filtering in search/list commands
