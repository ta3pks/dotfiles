**Goal:** Create line-delimited JSON interface for shell integration

## Context

- Shell pipelines
- Aider integration
- Simple scripting
- Testing/debugging

<task id="3" type="implementation">
<title>Create stdio main loop</title>

<action>
Create `memory/src/stdio/index.ts`:

1. Use readline to read stdin line by line
2. Parse each line as JSON
3. Route to appropriate handler
4. Output JSON response on stdout
5. Handle malformed JSON gracefully
6. Exit code 0 on success, 1 on error

```typescript
import * as readline from "readline";
import { MemoryService } from "../core/service.js";
import { handleCommand } from "./commands.js";

const rl = readline.createInterface({ input: process.stdin });
const service = new MemoryService();

rl.on("line", async (line) => {
  try {
    const request = JSON.parse(line);
    const response = await handleCommand(service, request);
    console.log(JSON.stringify(response));
  } catch (error) {
    console.log(JSON.stringify({ success: false, error: error.message }));
  }
});
```
</action>
